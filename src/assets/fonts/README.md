# Font Assets

Place your WOFF2 font files here:

- `primary-regular.woff2` — Regular (400) weight
- `primary-medium.woff2` — Medium (500) weight
- `primary-bold.woff2` — Bold (700) weight

Recommended fonts for a bold, modern design:
- Inter
- Outfit
- Plus Jakarta Sans
- Space Grotesk

Ensure the font includes Latin Extended characters for Spanish (ñ, á, é, í, ó, ú, ü).
