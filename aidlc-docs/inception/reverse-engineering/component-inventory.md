# Inventario de Componentes

## Aplicación
- **App root**: composición shell y accesibilidad.
- **Páginas**: `home`, `about`, `features`, `gallery`, `contact`.
- **Layout**: `header`, `footer`, `mobile-menu-drawer`.
- **Shared components**: `theme-toggle`, `language-switcher`, `skip-nav`, `back-to-top`.
- **UI kit interno**: `button`, `card`, `tabs`, `accordion`, `modal`, `form-field`, `badge`, `spinner`, `section-divider`.

## Servicios
- `theme.service.ts`
- `scroll.service.ts`
- `seo.service.ts`
- `whatsapp.service.ts`
- `animation.service.ts`

## Infraestructura de Proyecto
- **Tipo**: SPA estática desplegada en GitHub Pages.
- **Build**: Angular CLI + npm scripts.
- **Activos**: fuentes locales WOFF2 e imágenes optimizadas.
